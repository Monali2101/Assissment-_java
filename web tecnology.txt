Introduction to HTML

HTML (Hypertext Markup Language) is the standard language used to create web pages.
It defines the structure and layout of a webpage using tags.

Basic Structure of an HTML Document
<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
</head>
<body>
    Content of the webpage
</body>
</html>

<form>: Form tag for user input.
 Used to collect user input such as name, email, password etc.

<table>: Table tag for data representation.
 Used to display data in rows and columns.

<img>: Image tag for embedding images. 

 Used to embed images on a webpage.

List tags: <ul>, <ol>, and <li>. 

 Used to display items in list form.

<p>: Paragraph tag. 

 Used to write paragraphs of text.

<br>: Line break. 

 Used to break a line without starting a new paragraph.

<label>: Label for form inputs.

 Used to define a label for form input fields.

Q3) Overview of CSS and its importance in web design. 
Types of CSS:
	Inline CSS: Directly in HTML elements. 
	Internal CSS: Inside a <style> tag in the head section. 	External CSS: Linked to an external file.

| Type of CSS | Scope          | Priority | Usage          |
| ----------- | -------------- | -------- | -------------- |
| Inline      | Single element | High     | Quick changes  |
| Internal    | Single page    | Medium   | Small projects |
| External    | Multiple pages | Low      | Large websites |

Q5) Definition and difference between margin and padding. 
	How margins create space outside the element, and 	padding creates space inside.

Margin	Padding
Space outside the element	Space inside the element
Creates distance between elements	Creates space between content and border
Does not affect background color	Background color extends into padding

Margin is the space outside an HTML element. It creates distance between the element and other elements around it.

Controls outer spacing

Transparent area

Used to separate elements

Example: Space between two boxes

Definition of Padding

Padding is the space inside an HTML element, between the content and its border.

Controls inner spacing

Increases the size of the element

Improves readability

Example: Space between text and border
